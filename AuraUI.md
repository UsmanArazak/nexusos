# AuraUI Design System — NexusOS Official Reference

This document defines the official design language, **AuraUI**, for the entire NexusOS platform. Every module, component, and page built for NexusOS must strictly adhere to these specifications.

## 🎨 Color Palette

| Element | Color Code | Description |
| :--- | :--- | :--- |
| **Primary (Accent)** | `#7C3AED` | Purple: Used for active states, branding, and primary buttons. |
| **Primary Hover** | `#6D28D9` | Darker Purple: Used for button hover states. |
| **Sidebar Background**| `#FFFFFF` | White: Clean, minimal sidebar background. |
| **Main Background** | `#F3F4F6` | Light Grey: Page-level background color. |
| **Card Background** | `#FFFFFF` | White: Standard container background. |
| **Text Primary** | `#1C1C1C` | Dark Charcoal: Main body and heading text. |
| **Text Muted** | `#6B7280` | Grey: Descriptions, labels, and secondary text. |

### Stat Card Pastel Tints
*   **Students:** Light Purple (`#EDE9FE`)
*   **Staff:** Light Blue (`#DBEAFE`)
*   **Classes:** Light Peach (`#FFEDD5`)
*   **Fees/Success:** Light Green (`#DCFCE7`)

## 🧭 Sidebar & Navigation

*   **Background:** Solid White (`#FFFFFF`).
*   **Active State:** Purple text (`#7C3AED`) with a matching background tint and a **left border indicator** (accent bar).
*   **Logo:** NexusOS logo in Purple at the top left.
*   **Icons:** Minimalist SVG icons beside each navigation label.

## 🗂 Components

### Cards
*   **Style:** White background, subtle border (`#E5E7EB`), soft border radius.
*   **Shadows:** Low-opacity soft shadows for depth.
*   **Stat Cards:** Include colored icon badges (e.g., Graduation Cap for Students, Person for Staff).
*   **Empty States:** Centered icon with a friendly message and a clear Call to Action (CTA).

### Typography
*   **Labels:** Muted grey, uppercase/small-caps where appropriate.
*   **Values:** Bold, dark charcoal text for high visibility.

### Buttons
*   **Primary:** Purple background (`#7C3AED`), white text, rounded corners.
*   **Hover:** Transitions to a slightly darker shade (`#6D28D9`).

### Tables
*   **Styling:** Clean rows, subtle hover states, minimal borders.
*   **Empty State:** Clear messaging with placeholder graphics/icons.

## 📐 Layout Principles

*   **Whitespace:** Prioritize generous spacing to prevent clutter.
*   **Responsiveness:** Fluid layouts that scale gracefully from mobile to desktop.
*   **Consistency:** "AuraUI" is the source of truth—avoid ad-hoc color or spacing choices.
